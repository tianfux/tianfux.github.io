车队到码头做小票相关说明

发表于 2018-03-06

基础说明

  1. 无论真实货名是什么，车队需按照保函上货名答复码头，避免我司虚假货名暴露。
  2. 超期小票，需带原小票才能受理。
  3. 原小票丢失的，需另附小票补办申请，以及原计划号（原计划号可以找张爱鑫查询）

码头业务办理流程

  • 缴纳港杂码（大柜400元/箱，小柜260元/箱）
    1.车队自己的委托书（张爱鑫提供模板，需换成车队自己的名头）
    2.信风保函，另附箱明细。（加盖信风业务章）
  • 缴纳超期费（大柜75元/箱/期，小柜50元/箱/期，另附小票补打费20元/单号/次）
    1.车队自己的委托书（张爱鑫提供模板，需换成车队自己的名头）
    2.信风保函，另附箱明细。（加盖信风业务章）
    3.原小票
  • 小票补办（20元/单号/次）
    1.车队自己的委托书（张爱鑫提供模板，需换成车队自己的名头）
    2.信风保函，另附箱明细。（加盖信风业务章）
    3.提供原计划小票（找张爱鑫查询）
  • 冷柜小票
    1.车队自己的委托书（张爱鑫提供模板，需换成车队自己的名头）
    2.信风保函，另附箱明细。（加盖信风业务章）
  • 直提、预提（缴纳港杂另加100元直提/预提费用）
    1.车队自己的委托书（张爱鑫提供模板，需换成车队自己的名头）
    2.信风保函，另附箱明细。（加盖信风业务章）
    3.直、预提申请（含箱信息、车号、司机电话等），一式两份，一份办理业务，一份交中控室）

注意事项

  1. 需车队在集团码头有客户档案登记，即可在码头交钱开发票，如果没有需要去13楼进行登记（带营业执照复印件、开户行信息、一般纳税人资格证复印件，带公章，填表登记）。
  2. 目前我司系统未完善，每日车队垫付资金，箱明细发张爱鑫备份，以便后期核对。
  3. 我司协议标记重柜：啤酒（3期免堆）、化肥（3期免堆）、陶土（1期免堆）、小麦（3期免堆），提供给码头的明细，货名请根据协议标记填写。
  4. 公司业务委托书需拿车队自己的，如果没有我司可以发电子版空白委托书给车队，需要车队自己添加业务办理人身份证信息及图片，并盖自己车队的公章，开6点发票。
  5. 超期小票一定要拿原小票，原小票丢失，需另附小票补办申请，以及原计划号（原计划可以来找张爱鑫）

办理地点

  • 业务办理地点，1楼大厅，14楼中控室。

费用说明

上文已将各费用明细标明，若车队办单产生额外费用自理。费用金额纠纷，找张爱鑫咨询。
